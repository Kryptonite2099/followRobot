# json参数对照

## 必需参数
|      名称      |   类型   |     说明     |
| ------------- | -------- | ----------- |
|   target_url  |  string  |   目标网址   |
|    priority   |    int   |    优先级    |

## 可选参数
|      名称      |   类型   |       说明        |
| ------------- | -------- | ---------------- |
| require_login |  boolean |    是否需要登录    |
|   custom_re   |  boolean |  自定义页面匹配规则 |
|  article_tag  |  string  |   文章的html标签   |
